# Real-Time Stock Analytics Pipeline

A data engineering project that fetches, processes, and analyzes stock market data.

## 🎯 What Does This Project Do?

This pipeline will:
1. Fetch daily stock data from Yahoo Finance
2. Process and clean the data
3. Calculate technical indicators (moving averages, etc.)
4. Store data for analysis
5. Automate everything with Apache Airflow

## 🛠️ Technologies

- **Python** - Main programming language
- **Apache Airflow** - Workflow orchestration
- **PySpark** - Data processing
- **Docker** - Containerization
- **Databricks** - Cloud platform (later)
